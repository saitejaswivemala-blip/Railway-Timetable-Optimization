# Railway Timetable Optimization

Railway timetable optimization using a hybrid Genetic Algorithm (GA) and Frank-Wolfe (FW) approach.
Compared different optimization techniques like PSO, Genetic Algorithm, Simulated Annealing, hybrid Genetic Algorithm (GA) and Frank-Wolfe (FW)

## Features

- Train timetable optimization
- Passenger flow assignment
- Dynamic fare pricing
- Railway profit maximization
- Passenger cost minimization

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
